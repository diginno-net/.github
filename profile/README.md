# Diginno

**Practical Lark systems, workflow automation, and AI operations for growing SMEs.**

Diginno is a Vietnam-based implementation studio that helps businesses build better operating systems with **Lark**, **n8n**, **integrations**, and **AI-assisted workflows**.

## What we build

- **Lark / LarkBase systems**  
  CRM, HRM, finance operations, internal tools, approval flows, and business data structures

- **n8n workflow automation**  
  Production workflows that sync systems, reduce manual work, and improve data quality

- **AI operations**  
  AI agents and assisted workflows with human approval, auditability, and practical business use cases

- **Dashboards & integrations**  
  Web dashboards, API bridges, and operational tooling across business systems

## Why Diginno

- Deep specialization in **Lark** and operational system design
- Strong execution in **n8n automation** and multi-system integration
- Business-first approach: process, KPI, and operational fit before tools
- Production mindset: approvals, logs, permissions, and maintainability matter

## How we work

1. Understand the business process first
2. Design the data model and workflow
3. Implement automation and integrations
4. Add AI where it creates real operational value
5. Support rollout, training, and iteration
